Packagecom.pblabs.box2D
Classpublic class Box2DSpatialComponent
InheritanceBox2DSpatialComponent EntityComponent
ImplementsIMobileSpatialObject2D

Wraps a b2Body for use with a Box2DManagerComponent and integration into PushButton Engine.

Most properties map directly to actions in Box2D, so most of the time your questions about this component will really be Box2D questions in disguise.
