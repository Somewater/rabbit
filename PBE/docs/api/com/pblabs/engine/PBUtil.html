public static function duckAssign(source:Object, destination:Object, abortOnMismatch:Boolean = false):void

Assigns parameters from source to destination by name.

This allows duck typing - you can accept a generic object (giving you nice {foo:bar} syntax) and cast to a typed object for easier internal processing and validation.

Parameters
source:Object — Object to read fields from.
 
destination:Object — Object to assign fields to.
 
abortOnMismatch:Boolean (default = false) — If true, throw an error if a field in source is absent in destination.
