Classpublic class InputManager

ImplementsITickedObject

The input manager wraps the default input events produced by Flash to make them more game friendly. For instance, by default, Flash will dispatch a key down event when a key is pressed, and at a consistent interval while it is still held down. For games, this is not very useful.

The InputMap class contains several constants that represent the keyboard and mouse. It can also be used to facilitate responding to specific key events (OnSpacePressed) rather than generic key events (OnKeyDown).
