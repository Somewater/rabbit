Classpublic class ThinkingMachineInstance
InheritanceThinkingMachineInstance ThinkingComponent EntityComponent
ImplementsIMachine

A state machine which shares its description with other machines. Useful when you have many identical FSMs and don't want to pay memory/initialization overhead for each one. Use Machine if you want a one-off FSM.
